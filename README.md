# Operating Room Scheduling Optimization

BME 411 Final Project to optimize OR scheduling based on [A constraint-programming-based branch-and-price-andcut approach for operating room planning and scheduling (Doulabi et. al)](https://hanalog.ca/wp-content/uploads/2016/09/HPR_ORSched_Joc.pdf)

 Running Code

1. Create and activate python venv

```bash
python -m venv or-env
source or-env/bin/activate
pip install -r requirements.txt
```

2. Run code

For dummy run:
```bash
python run_optimization_scenario.py

```

For test cases:

```bash
python test_cases.py
```
